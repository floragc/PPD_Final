## Objectif de cette application

De gros volumes de documents scientifiques sont maintenant disponibles en libre accès. Par exemple, la base Pubmed propose un dump XML de 25 millions d’abstracts médicaux. L’objectif du projet est de charger ces documents dans une base hébergée sur le Cloud et de les rendre accessibles à travers une interface conviviale.

## Outils mis en oeuvre
* Base de données full-text hébergée sur le Cloud Microsoft Azure
* Site Web Node.js hébergé sur le Cloud Microsoft Azure, framework Express et moteur de template Jade
